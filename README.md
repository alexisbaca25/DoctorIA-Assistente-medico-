# DoctorIA - Sistema Experto (Proyecto)

Estructura del proyecto:
```
DoctorIA/
├── main.py
├── interfaz.py
├── motor_inferencia.py
├── base_conocimiento.py
├── recomendaciones.py
├── utils.py
└── assets/
    └── logo.png
```

Instrucciones:
1. Asegúrate de tener Python 3.8+ instalado.
2. Ejecuta: `python main.py`
3. Para exportar el diagnóstico a PDF, usa el botón "Exportar PDF".

Notas:
- Esta app es educativa y **no** sustituye a un profesional de la salud.
- El PDF se genera creando una imagen y guardándola como PDF (compatibilidad alta).
